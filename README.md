📁 Smart File Vault

Secure full-stack file upload and management system built using Angular and Node.js with stream-based file handling.

🚀 Tech Stack
Frontend
    Angular
    Angular Material
    TypeScript

Backend
    Node.js
    Express.js
    Stream-based uploads (Busboy)

✨ Features
    📤 File upload with real-time progress bar
    📁 Local file storage
    📊 Upload tracking using Angular HttpEvents

📂 Project Structure
    smart-file-vault/
    ├── frontend/    # Angular application
    ├── backend/     # Node.js API
    └── README.md

⚙️ Getting Started

1️⃣ Clone Repository
    git clone https://github.com/<your-username>/smart-file-vault.git
    cd smart-file-vault

Backend Setup
    cd backend
    npm install
    npm run start

Runs on:
    http://localhost:4000

Frontend Setup
    cd frontend
    npm install
    ng serve

Runs on:
    http://localhost:4200
